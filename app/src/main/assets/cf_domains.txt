# Open-site fallback domains.
#
# When a normal range scan finds zero reachable Cloudflare IPs, these domains
# are resolved to their Cloudflare edge IPs and probed instead — the idea being
# that a site which still LOADS through Cloudflare on the user's network points
# at an edge address the network lets through.
#
# Lines starting with '#' are ignored. One host per line. Users can add their
# own (e.g. a site they know is open right now) from the app's advanced settings.
